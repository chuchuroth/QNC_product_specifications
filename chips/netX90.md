**netX 90** 并不是一个通信协议，而是德国赫优讯（Hilscher）公司推出的一款**高度集成的多协议工业通讯SoC（片上系统）芯片**。你可以把它理解为一个专门为工业设备设计的“万能翻译器”，能让一个设备通过一颗芯片，支持几乎所有的主流工业实时以太网和现场总线协议。

它最核心的设计理念是**高集成度**、**多协议灵活性**以及**内置的安全性**。

### 🧠 1. 核心架构：双核隔离，各司其职

netX 90 芯片内部集成了两个独立的ARM Cortex-M4处理器内核，各自承担不同的任务：

- **通讯核 (Communication Cortex-M4 @ 100 MHz)**：专门负责处理实时工业通信。它运行由赫优讯提供的、预认证好的各种协议栈固件，确保通信的实时性和稳定性，将开发者从复杂的协议细节中解放出来。
- **应用核 (Application Cortex-M4 @ 100 MHz)**：留给用户开发自己的应用程序，如设备控制、数据采集、算法实现等。这个内核带有FPU（浮点运算单元），性能更强。

这种物理隔离的设计优势很明显：即使通讯部分出现拥塞或问题，也不会影响到核心的应用功能。同时，从安全角度看，这种隔离也防止了外部攻击通过通信接口直接渗透到应用层。

### 🔌 2. 最大亮点：无与伦比的协议灵活性

这是 netX 90 最强大的特性。通过加载不同的固件，同一颗芯片可以支持数十种工业通信协议，这极大地减少了硬件库存和备货风险。

| 协议类型 | 支持的协议举例 |
| :--- | :--- |
| **实时以太网** | PROFINET, EtherNet/IP, EtherCAT, POWERLINK, Sercos, CC-Link IE Field Basic |
| **现场总线** | PROFIBUS, CANopen, DeviceNet, CC-Link |
| **工业物联网 (IIoT)** | OPC UA, MQTT |

它支持的协议还在不断更新，并且已经通过了PROFINET和EtherNet/IP的官方一致性认证，确保了产品的兼容性。

### 🔒 3. 安全特性：为工业4.0设计

针对日益严峻的网络安全形势，netX 90 内置了丰富的硬件安全功能，以支持IEC 62443标准（工业自动化和控制系统网络安全标准）：

- **安全启动 (Secure Boot)**：确保芯片启动时加载的每一个固件都是经过授权和验证的，防止恶意代码注入。
- **硬件加密加速器**：支持AES-256, RSA-4096, ECC-512等现代加密算法，能够高效地处理SSL/TLS加密通信，而不会给CPU带来过重负担。
- **内置防火墙**：可以在硬件级别对通信数据进行过滤和隔离。

### 📦 4. 高集成度与封装

netX 90 的尺寸仅为 **10mm x 10mm**，采用144针BGA封装。它的高集成度体现在内部集成了许多原本需要外部单独实现的组件，例如：

- **双端口以太网PHY**：可以直接连接带隔离变压器的RJ45接口，无需外接PHY芯片，简化了硬件设计。
- **DC/DC转换器和上电复位电路**：进一步减少了外部元器件数量，降低了BOM（物料清单）成本和PCB空间。

### 💡 5. 典型应用场景

基于上述特性，netX 90非常适合于那些对空间、成本、功耗和协议灵活性有严苛要求的从站设备，例如：

- **小型现场设备**：传感器、编码器、阀门岛、IO-Link主站。
- **驱动与控制**：伺服驱动器、变频器、电机控制器（赫优讯为此提供了专门的netMOTION解决方案）。
- **网关与协议转换器**：利用其多协议能力，在不同工业网络间桥接数据。

### 🛠️ 6. 两种工作模式

netX 90可以根据你的需求，以两种模式工作：

1.  **单芯片模式**：同时使用内部的“应用核”和“通讯核”，芯片自身就完成了所有工作。适用于对成本和空间要求极高的简单设备。
2.  **协处理器模式**：只使用芯片的“通讯核”，外部连接一个主控CPU（如STM32）。netX 90作为专门的通信协处理器，负责处理复杂的工业网络协议，而主CPU专注于运行复杂的应用逻辑。这是目前比较主流的用法。

总的来说，**netX 90是一个为工业设备赋予“联网说话”能力的强大SoC芯片**，它把复杂的多协议支持、高实时性通信和硬件级的安全性都集成在了一个小封装里，极大地简化了工业设备的网络化开发。
